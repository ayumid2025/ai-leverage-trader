# AI Leverage Trader

An AI trading agent that simulates trading with **$10 initial capital and 100x leverage**.  
⚠️ **For educational purposes only. Do not use with real money.**

## Features (to be implemented)
- [ ] Fetch real stock data
- [ ] Simulate margin account with leverage
- [ ] Implement trading strategy (e.g., trend following)
- [ ] Backtest with historical data
- [ ] Risk management (stop-loss, position sizing)
- [ ] Paper trading simulation

## Project Structure
- `src/` - source code
- `data/` - downloaded data
- `notebooks/` - Jupyter notebooks for analysis
- `config/` - configuration files
- `tests/` - unit tests

## Setup
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run backtest: `python src/backtester.py`
